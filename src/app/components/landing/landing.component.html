SmartChart

Welcome to the SmartChart platform. SmartChart provides a system for multiple client applications to build upon a common backend service built around HL7 FHIR. This client focuses on patient chart abstraction to enable the completion of patient related forms.

To begin, please select the "Form Manager" option in the left navigation bar. From there you may load an active form or start a backend job to pull patient data for a new form.
